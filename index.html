Dumpling logo - polynomial function

Parent Function

y=x^4

Transformed Equation

y=-4(x-0.98)^4-2.78(x-0.789)^2+1.124

Transformation

I chose a quartic polynomial function because the dumpling logo in the photograph had a smooth curved shape with a rounded top. I reflected the graph across the x-axis so the curve opened downward like the dumpling logo in the image. I then translated the graph 0.98 units to the right and 1.124 units upward so the graph aligned with the logo’s position. A vertical stretch by a factor of 4 was applied to better match the curvature of the dumpling shape, and an additional quadratic term was added to make the curve smoother and more realistic.


Summary of Transformations

  • Reflection across the x-axis
  • Horizontal translation 0.98 units to the right
  • Vertical translation 1.124 units upward
  • Vertical stretch by a factor of 4
  • Additional quadratic term added for a smoother curve

Domain and Range

The restricted domain of the function is: 0.65 ≤ x ≤ 0.95 The graph was restricted so that it only modeled the visible portion of the dumpling logo in the photograph. The approximate range of the function is: 1.02 ≤ y ≤ 1.12

Raised Arm – Tangent Function

Parent Function

y = tan(x)

Transformed Equation

y = tan(0.35x)/4 + 2

Transformations

I selected the tangent parent function because the shape of my raised arm closely resembled the curve of a tangent graph. To better fit the image, I applied a horizontal stretch to make the graph wider and smoother. A vertical compression was also used to reduce the steepness of the curve. Finally, the graph was shifted upward so the curve aligned more closely with the position of my arm in the photo.

Domain and Range

The domain of the function is restricted to -3 < x < 4 because only the visible portion of the arm is being modeled. Restricting the domain removes unnecessary repeating sections and asymptotes from the tangent graph.

Domain: -3 < x < 4

Range: All real numbers
